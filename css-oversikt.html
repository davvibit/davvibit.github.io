CSS-kodesnutter

Her er en liste over grunnleggende CSS-egenskaper med korte forklaringer og eksempler på gyldige verdier.

Elementvelger (f.eks., h1, p)

Velger alle forekomster av et element.

p { color: red; }
Klassevelger (f.eks., .className)

Velger elementer med en spesifikk klasse.

.highlight { background-color: yellow; }
ID velger (f.eks., #idName)

Velger et enkelt element med en spesifikk ID.

#unique { font-size: 20px; }
color

Setter fargen på teksten.

color: blue;
color: #ff0000;
color: rgb(0, 255, 0);
font-family

Setter skrifttypen for teksten.

font-family: Arial, sans-serif;
font-family: 'Times New Roman', serif;
font-size

Setter størrelsen på teksten.

font-size: 16px;
font-size: 1.5em;
font-size: 120%;
font-weight

Setter tykkelsen (fetheten) på teksten.

font-weight: normal;
font-weight: bold;
font-weight: 700;
text-align

Setter justeringen av teksten.

text-align: left;
text-align: center;
text-align: right;
margin

Setter ytterkanten av elementer.

margin: 10px;
margin: 5px 10px 15px 20px;
margin-top: 10px;
padding

Setter innerkanten av elementer.

padding: 10px;
padding: 5px 10px 15px 20px;
padding-left: 10px;
border

Setter kanten (rammen) til elementer.

border: 1px solid black;
border-width: 2px;
border-color: blue;
width

Setter bredden på elementer.

width: 100px;
width: 50%;
height

Setter høyden på elementer.

height: 200px;
height: 50vh;
background-color

Setter bakgrunnsfargen til et element.

background-color: yellow;
background-color: #00ff00;
background-color: rgba(255, 0, 0, 0.5);
background-image

Setter et bakgrunnsbilde for et element.

background-image: url('image.jpg');
background-image: linear-gradient(to right, red, blue);
